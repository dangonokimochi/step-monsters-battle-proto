# Step Monsters - Battle Proto

Step Monstersの戦闘システムプロトタイプ。

## 概要

「散歩×モンスター収集×グリッドSRPG×ローグライト」ゲームの戦闘部分を検証するプロトタイプ。

- 8×8グリッド上の4vs4ターン制バトル
- SPD順に1体ずつ行動
- 射程・射線・地形による戦術的な位置取り
- React + TypeScript

## セットアップ

```bash
npm install
npm run dev
```
